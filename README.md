# AOC solutions

## General conventions

### Variables in capitalized letters
Things that should not change, usually intermediate data structures to store input conveniently.

  - `D` : Data
  - `L` : Lines
  - `R` : Results
  - `G` : Graph

### Variables in lower case

  - `i`, `j`, `k` : counters
  - `r` : result
  - `u`, `v`, `w` : vectors
  - `co` : column
  - `li` : line
